C++ es un lenguaje de programación orientado a objetos de propósito general que es una extensión del lenguaje de programación C. Fue desarrollado por Bjarne Stroustrup en 1983, con el objetivo de agregar funciones orientadas a objetos a C, manteniendo la eficiencia y el control de un lenguaje de bajo nivel. C ++ se usa ampliamente en muchas áreas de desarrollo de software, incluida la programación de sistemas, el desarrollo de juegos y las aplicaciones de escritorio.


Algunas de las características clave de C++ incluyen:
  • Clases y Objetos.

  • Encapsulación, herencia y polimorfismo.

  • Plantillas y programación genérica.

  • Manejo de excepciones.

  • Sobrecarga de operadores.

    Productos comerciales de C++



    MOZILLA


    Mozilla usa un subconjunto de C++. Se requiere C++ 14 para compilar Mozilla 59, Mozilla Firefox y el cliente de chat por correo electrónico Thunderbird están escritos con C++.

    MONGODB


    MongoDB es una base de datos de código abierto, ampliamente utilizada como almacén de back-end para aplicaciones web, así como en grandes empresas como Viacom, gigantes de la biotecnología y Disney.

    Compiladores


    Se sabe que el lenguaje C++ es un lenguaje compilado, esta es la razón principal por la que la mayoría de los compiladores están escritos principalmente en lenguaje C++ únicamente. Los compiladores utilizados para compilar otros lenguajes como C# , Java , etc. están principalmente escritos solo en C++. También se utiliza en el desarrollo de estos lenguajes, ya que C++ es independiente de la plataforma y puede crear una variedad de software.

    MICROSOFT


    Muchas de las aplicaciones de Windows que usa regularmente están escritas en C++. Cuenta con herramientas para desarrollar y depurar código C++, especialmente código escrito para DirectX, API de Windows y .NET .
